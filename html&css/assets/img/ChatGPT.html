Untuk memberikan jarak antara bagian judul dan bagian di bawahnya, Anda bisa menggunakan properti CSS seperti margin-bottom atau padding-bottom pada elemen pembungkus judul. Berikut adalah beberapa cara yang dapat dilakukan:

1. Tambahkan Margin pada Elemen Judul

Gunakan kelas CSS untuk elemen judul atau elemen pembungkus seperti <nav>.

html
<style>
    .navbar-top {
        margin-bottom: 20px; /* Atur jarak sesuai kebutuhan */
    }
</style>

2. Tambahkan Margin Langsung pada Elemen HTML

Jika Anda tidak ingin mengedit file CSS, tambahkan atribut style langsung pada elemen <nav>.

html
<nav class="navbar navbar-top navbar-expand navbar-dashboard navbar-dark ps-0 pe-2 pb-0" style="margin-bottom: 20px;">
    ...
</nav>

3. Gunakan Kelas Bootstrap

Jika Anda menggunakan Bootstrap, manfaatkan utilitas mb-* (margin-bottom) untuk memberikan jarak.

html
<nav class="navbar navbar-top navbar-expand navbar-dashboard navbar-dark ps-0 pe-2 pb-0 mb-4">
    ...
</nav>

4. Tambahkan Padding di Bagian Bawah Elemen Judul

Jika lebih cocok menggunakan padding, Anda bisa menambahkannya sebagai berikut:

html
<style>
    .navbar-top {
        padding-bottom: 20px; /* Atur jarak sesuai kebutuhan */
    }
</style>

Pilih salah satu metode di atas sesuai kebutuhan Anda. Biasanya, menggunakan utilitas Bootstrap seperti mb-4 sudah cukup praktis.

Untuk mengganti bagian "PENARIKAN DATA JM" dengan gambar jasamarga.jpg yang sesuai dengan lebar navbar, Anda bisa mengganti elemen HTML terkait dan menambahkan beberapa aturan CSS. Berikut langkah-langkahnya:

HTML

Ubah elemen <li> yang berisi teks "PENARIKAN DATA JM" menjadi seperti ini:

html
<li class="nav-item">
    <a href="#" class="nav-link d-flex align-items-center">
        <img src="/path/to/jasamarga.jpg" class="w-100 img-fluid" alt="Jasa Marga">
    </a>
</li>

CSS

Tambahkan beberapa aturan CSS untuk memastikan gambar memenuhi lebar navbar dan mempertahankan proporsi:

html
<style>
    .nav-item img {
        width: 100%; /* Gambar memenuhi lebar navbar */
        height: auto; /* Menjaga rasio aspek gambar */
        object-fit: cover; /* Memastikan gambar tidak terdistorsi */
    }
</style>

Penjelasan:

  1. <img> dalam HTML:

    • Atribut src mengarah ke lokasi file jasamarga.jpg.
    • Atribut alt digunakan untuk memberikan teks alternatif jika gambar tidak bisa ditampilkan.

  2. CSS:

    • width: 100%; memastikan gambar memenuhi lebar elemen pembungkusnya.
    • height: auto; menjaga rasio aspek gambar sehingga tidak terdistorsi.
    • object-fit: cover; memastikan gambar tetap terlihat baik jika ukurannya dipotong.

Hasil

Dengan pengaturan di atas, gambar akan menggantikan teks "PENARIKAN DATA JM" dan akan otomatis menyesuaikan dengan lebar navbar. Pastikan mengganti /path/to/jasamarga.jpg dengan path aktual tempat file jasamarga.jpg disimpan.
